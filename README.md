# Extracción Forense de Datos en iOS

## Descripción

Este trabajo consiste en realizar una **extracción lógica de información** de un dispositivo iOS y analizar los datos obtenidos.

La extracción lógica usa los datos accesibles mediante el sistema operativo, como contactos, mensajes o registros, sin hacer una copia completa de la memoria del dispositivo.

## Objetivo

Aprender el proceso básico de análisis forense en iPhone y documentar la información de forma ordenada.

## Conclusión

La práctica permite entender cómo se extrae y revisa información de un móvil iOS de manera sencilla y controlada.
